1.本周安全漏洞级别概况

据2018年7月5日的扫描情况，共发现400个漏洞, 其中高风险漏洞18个，占总漏洞的4.50%; 中风险漏洞96个,占总漏洞的24.00%; 低风险漏洞286个，占总漏洞的71.50%.,以下为最新发布的十条高风险漏洞:

CVE-2018-10730 2018-06-19 All Phoenix Contact managed FL SWITCH 3xxx, 4xxx, 48xx products running firmware version 1.0 to 1.33 are prone to OS command injection.
CVE-2018-10123 2018-06-19 p910nd on Inteno IOPSYS 2.0 through 4.2.0 allows remote attackers to read, or append data to, arbitrary files via requests on TCP port 9100.
CVE-2018-11013 Exec Code Overflow 2018-06-19 Stack-based buffer overflow in the websRedirect function in GoAhead on D-Link DIR-816 A2 (CN) routers with firmware version 1.10B05 allows unauthenticated remote attackers to execute arbitrary code via a request with a long HTTP Host header.
CVE-2018-11032 Sql 2018-06-19 PHPRAP 1.0.4 through 1.0.8 has SQL Injection via the application/home/controller/project.php search() function.

2.漏洞类别

本次所扫描到的漏洞可根据风险类别分为以下主要10类, 高危漏洞类别按数量正序分布如下： Not Defined: 147个； XSS: 50个； Exec Code: 34个； DoS: 33个； Overflow: 23个； CSRF: 15个； DoS Overflow: 14个； Bypass: 13个； Sql: 12个； +Info: 10个。

3.高风险且未发布补丁的漏洞

cveId Vulnerability Type Date 描述
CVE-2018-10123 2018-06-19 Inteno IOPSYS是瑞典Inteno Broadband技术公司的一套开放式服务交付平台。该平台由网关操作系统、家庭门户和多种软件开发工具包组成。p910nd是其中的一个打印守护程序。 Inteno IOPSYS 2.0版本至4.2.0版本上的p910nd存在安全漏洞。远程攻击者可通过在TCP 9100端口上发送请求利用该漏洞读取任意文件或向任意文件添加数据。
CVE-2018-11013 Exec Code Overflow 2018-06-19 D-Link DIR-816 A2是友讯（D-Link）公司的一款无线路由器产品。GoAhead是其中的一款嵌入式Web服务器。 使用1.10B05版本固件的D-Link DIR-816 A2(CN)中的GoAhead的‘websRedirect’函数存在栈缓冲区溢出漏洞。远程攻击者可通过发送带有较长HTTP Host包头的请求利用该漏洞执行任意代码。
CVE-2018-11032 Sql 2018-06-19 PHPRAP是一套基于PHP的轻量级开源文档管理系统。 PHPRAP 1.0.4版本至1.0.8版本中存在SQL注入漏洞。远程攻击者可借助application/home/controller/project.php文件中的‘search()’函数利用该漏洞执行任意的SQL命令。
CVE-2018-10682 Exec Code 2018-06-18 Red Hat Wildfly（前称JBoss Application Server）是美国红帽（Red Hat）公司的一款基于JavaEE的开源应用服务器。 Red Hat WildFly 10.1.2.Final版本中存在安全漏洞。远程攻击者可利用该漏洞访问TCP 9990端口上的管理面板，部署恶意的.war文件，执行代码。
CVE-2018-10683 2018-06-18 Red Hat Wildfly（前称JBoss Application Server）是美国红帽（Red Hat）公司的一款基于JavaEE的开源应用服务器。 Red Hat WildFly 10.1.2.Final版本中存在安全漏洞。攻击者可利用该漏洞不经身份认证访问服务器。
